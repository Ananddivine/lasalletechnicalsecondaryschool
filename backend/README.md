# Admissions Backend Scaffold

This folder contains a lightweight mock backend for the admissions admin portal.

## Endpoints

- `GET /api/health`
- `POST /api/admin/login`
- `GET /api/admission-results`
- `GET /api/admission-results/:id`

## Default Login

- Email: `admin@lasalletech.edu.pg`
- Password: `Admin@2026`

## Run

```bash
cd backend
npm start
```

The server uses only built-in Node.js modules, so no extra install step is required for this scaffold.
